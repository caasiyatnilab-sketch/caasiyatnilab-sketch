### About Me

I'm a full-stack developer focused on shipping real products -- web apps, REST APIs, AI-powered tools, and automation systems. I write code that works in production, not just in tutorials.

Currently building with **React**, **Node.js**, **Express**, **MongoDB**, **Docker**, and integrating multiple **AI providers** (Groq, OpenRouter, HuggingFace, Ollama). I also maintain a fleet of **23 autonomous GitHub bots** that handle CI, security, deployment, and code quality across all my repos.
